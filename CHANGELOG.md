# Changelog

## [1.0.0] - 2026-04-27

### Added
- Initial release of Image-Charts Generic Chart Generator PCF component
- Support for all Image-Charts chart types via `chartType` property
- Support for raw chart data via `chartData` property
- Support for custom parameters via `customParams` property
- Support for Enterprise mode (HMAC-SHA256 signing)
- Support for Private Cloud mode (custom domain)
- Debounced updates (300ms)
- Retry with exponential backoff (15s timeout)
- Error placeholder support
- Debug URL display
